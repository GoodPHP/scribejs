Simple, Intuitive API

      

        

          
editor.bold()

          
Direct methods for formatting — no command strings to memorize.

          editor.italic() | editor.underline()
        

        

          
editor.heading(2)

          
Block formatting with simple, typed parameters.

          editor.blockquote() | editor.orderedList()
        

        

          
editor.link(url)

          
Insert and manage links with ease.

          editor.link('https://...') | editor.unlink()
        

        

          
editor.on('change', fn)

          
Subscribe to events for real-time updates.

          'change' | 'focus' | 'blur' | 'selectionChange'
